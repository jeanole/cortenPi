# CortenPi - Sheetmetal Woodfire Stove

This project contains the design files for a sheetmetal woodfire stove, optimized for making delicious pizza. The stove is constructed from plain laser-cut 3mm sheetmetal, requiring no bending or welding. It incorporates schamot stone for heat retention and optimal cooking performance.

## Features
- **Simple Construction**: Laser-cut design with no bending or welding required
- **Materials**: 3mm sheetmetal with schamot stone lining
- **Purpose**: Wood-fired pizza oven
- **Efficiency**: Designed for excellent heat retention and even cooking
